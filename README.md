# SOTD

Sherevo Of The Day

## REQUIREMENTS

 - *nix or BSD based OS
 - Ruby 3.x.x or higher

## INSTALL

```
$ ./installer.sh
```

## USAGE

You can see so called «‎sherevo» every time whem you launch terminal, but
you can execute the binary directly:
‎
```
$ sotd

PIZDEC SHEREVO!!1

```
